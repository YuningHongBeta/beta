# beta

Geant4 simulation repository for the E63 experiment detector studies.

## Contents

| Directory | Description |
|-----------|-------------|
| [beta_org/](beta_org/) | Beta-ray detector simulation with custom physics and biasing |
| [Cal_src/](Cal_src/) | Calorimeter (Ge calibration source) simulation |

## Quick start

Each sub-project builds independently with CMake + Geant4. See the README in each directory for details.
